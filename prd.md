
# Liquid Silk 产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品定位
**Liquid Silk** 是一款跨设备的动态壁纸应用，通过摄像头手势识别技术操控宇宙星空背景下的液态金属丝 3D 粒子效果。

### 1.2 核心价值
- 沉浸式视觉：宇宙星空 + 液态金属丝的高品质 3D 动态效果
- 自然交互：通过摄像头手势实现零接触操控
- 多屏联动：支持多设备实时同步

---

## 二、功能需求

### 2.1 核心功能矩阵

| 模块 | 功能描述 | 优先级 |
|------|----------|--------|
| 动态壁纸渲染 | 3D粒子系统 + 金属丝效果，星空背景 | P0 |
| 手势交互 | 滑动、捏合、旋转三种手势识别 | P0 |
| 多设备同步 | 双设备实时状态同步（200ms内） | P0 |
| 状态持久化 | 保存用户自定义参数 | P1 |

### 2.2 手势交互规格

| 手势类型 | 交互效果 | 阈值要求 |
|----------|----------|----------|
| 滑动 | 粒子流动方向跟随手指轨迹 | 移动距离 ≥ 10px |
| 捏合 | 粒子系统整体收缩/放大 | 缩放比例变化 ≥ 0.1 |
| 旋转 | 星空背景视角同步旋转 | 基于手势角度计算 |

---

## 三、非功能性需求

### 3.1 性能要求

| 指标 | 要求 |
|------|------|
| 帧率 | ≥ 60 FPS (正常), ≥ 50 FPS (降级) |
| 粒子数量 | ≥ 100,000 |
| 同步延迟 | ≤ 200 ms |
| 内存占用 | < 500 MB |

### 3.2 兼容性要求

| 平台 | 浏览器 | 支持状态 |
|------|--------|----------|
| macOS | Chrome/Firefox | 完全支持 |
| Windows | Chrome/Firefox/Edge | 完全支持 |
| iOS | Safari | 需测试 |
| Android | Chrome | 需测试 |

---

## 四、验收标准

- [ ] 摄像头正常调用，手势识别准确率 ≥ 85%
- [ ] 三种手势交互响应流畅，无卡顿
- [ ] 双设备同步延迟 ≤ 200ms
- [ ] 状态重启后可恢复
- [ ] 低配设备自动触发降级
